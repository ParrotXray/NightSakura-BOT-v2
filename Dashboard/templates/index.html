我們在裡面內建多種語言系統讓您選擇，目前支援英文、日文、繁/簡中文。

音樂功能-music

讓使用者播放音樂的指令，目前可解析youtube、bilibili的連結。

/ play 播放歌曲。

/ join 連接語音。

/ pasue 暫停當前正在播放的歌曲。

/ resume 繼續播放歌曲。

/ volume 顯示當前播放的音量及改變播放音量。

/ leave 離開語音頻道。

管理功能-manager

讓BOT來管理與紀錄伺服器的點點滴滴。

/ serverinfo 查看伺服器資訊。

/ userinfo 查看使用者資訊。

/ autorole 自動給予使用者所指定的身分組。

/ reactionroles 點擊表情使用者所指定的身分組。

問題與回報

您可以使用BOT的內建指令 /feedback 直接向我們回報您遇到的問題。
